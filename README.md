# 📊 أداة التحليل المالي المتكاملة

أداة تفاعلية لتحليل الأسهم باستخدام Python و Streamlit.

## المؤشرات المتاحة
- المتوسطات المتحركة (MA5, MA10, MA20, MA50, MA100)
- RSI, MACD, Stochastic
- Bollinger Bands, ATR
- مستويات فيبوناتشي
- إشارات شراء/بيع آلية

## التشغيل
```bash
pip install -r requirements.txt
streamlit run app.py
